An HTML5 form validation library in javascript.

Extending native HTML5 form validation to add more complex, customisable and consistent client-side validation.

Designed to make HTML5 form validation as simple as possible with declarative, readable, easily maintainable, extensible, accessible, reusable code.

No dependencies, supports all modern browsers and IE8 (standards mode)+.

Fork on github

What does it do?

  • Standardises client-side form validation error display and behaviour across browsers
  • Uses and encourages accessible markup
  • Create multi-step forms from a single form
  • Adds group validation for checkbox and radio groups
  • Supports real-time form validation
  • Supports autocomplete
  • Polyfills HTML5 validation for IE8 standards mode +
  • Supports first error inline (native browser-style), full form inline and linked list error messages
  • Customisable CSS hooks for all features and states
  • Group validation for custom groups of any field type
  • Minimum number of valid fields for any group
  • Customisable form completion callback
  • Customisable error messages by field and error type, and by individual field
  • Custom constraint validation - make your own constraint
  • Adds more form control allowing enabling of fields based on previous field completion

Check out the documentation for setup and configuration, and the examples for use cases.
