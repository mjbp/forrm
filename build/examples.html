4. Steps

6. Custom validation constraints

Define custom constraint functions, pass them to Forrm in the constructor and attach them declaratively to elements. In this example the value of both fields must first pass the usual constraint validation then must also be the same value as each other.

<form id="example-constraint" action="/">
  <fieldset>
    <div class="forrm-row">
      <label for="example-constraint-1">Password: *</label>
      <input type="password" id="example-constraint-1" name="example-constraint-group" value="" required aria-required="true">
    </div>
    <div class="forrm-row">
      <label for="example-constraint-2">Re-type your password: *</label>
      <input type="password" id="example-constraint-2" name="example-constraint-group" value="" required aria-required="true" data-forrm-custom-constraint="match-values">
    </div>
    <input type="submit" value="Submit">
  </fieldset>
</form>

<script src="js/forrm.min.js">
<script>
Forrm.init('#example-constraint', {customConstraint :
                                    {'match-values' : function () {
                                         var matcher = document.querySelector('#example-constraint-1').value,
                                             error = 'The passwords must be the same';
                                         if (this.value === matcher) {
                                           return '';
                                         } else {
                                           return error;
                                         }
                                       }
                                    }});
</script>

7. Custom fail and success functions

Pass custom functions to Forrm in the constructor to be executed on validation fail, or on validation success instead of form submission.

<form id="example-functions" action="/">
  <div class="forrm-row">
    <label for="example-functions">How old are you? *</label>
    <select id="example-functions" name="example-functions" required aria-required="true">
      <option value="">Select one…</option>
      <option value="18-35">18-35</option>
      <option value="36-55">36-55</option>
      <option value="55-85">55-85</option>
      <option value="85+">85+</option>
    </select>
  </div>
  <input type="submit" value="Submit">
</form>

<script src="js/forrm.min.js">
<script>
Forrm.init('#example-functions', {
            fail : function () {window.alert(':( Validation failed');},
            pass : function () {window.alert(':) Validation success');}
        });
</script>

8. Conditionals

Enable a disabled field or part of a form based on a previous field. The defaut conditional constraint is for the anticedent field to have a non-null value. This example uses a custom condtional constraint to enable the consequent field only if the value of the group equals 'other'.

<form id="example-conditional" action="/">
  <div class="forrm-row forrm-row-radio">
    <fieldset>
      <legend>I have a pet*</legend>
      <div class="forrm-row forrm-row-radio">
        <input type="radio" id="example-conditional-1" name="example-conditional-1" value="cat" required aria-required="true" data-forrm-conditional="isOther">
        <label for="example-conditional-1">Cat</label>
      </div>
      <div class="forrm-row forrm-row-radio">
        <input type="radio" id="example-conditional-2" name="example-conditional-1" value="dog" required aria-required="true" data-forrm-conditional="isOther">
        <label for="example-conditional-2">Dog</label>
      </div>
      <div class="forrm-row forrm-row-radio">
        <input type="radio" id="example-conditional-3" name="example-conditional-1" value="other" required aria-required="true" data-forrm-conditional="isOther">
        <label for="example-conditional-3">Other</label>
      </div>
    </fieldset>
  </div>
  <div class="forrm-row forrm-disabled isOther">
    <label for="test-conditional-4">If other, please specify*</label>
    <input type="text" id="test-conditional-4" name="test-conditional-4" value="" required aria-required="true" disabled>
  </div>
  <input type="submit" value="Submit">
</form>

<script src="js/forrm.min.js">
<script>Forrm.init('#example-conditional', {conditionalConstraint : {
            isOther : function () {
                return this.value === 'other';
            }
        }});</script>
