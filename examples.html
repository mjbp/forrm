1. Default installation

HTML5 validation with inline errors straight out of the box

//Create an HTML5 form
<form action="/search" method="post" id="forrm-search">
  <div class="forrm-row">
    <label for="search-field">Search*</label>
    <input type="search" id="search-field" name="search-field" placeholder="Your search terms..." value="" aria-required="true" required>
  </div>
  <input type="submit" value="Search">
</form>
//Include minfied version and instantiate form
<script src="js/forrm.min.js">
<script>Forrm.init('#forrm-search');

2. Linked list of errors

Display a list of errors at the top of a form linking to the invalid field.

//Create an HTML5 form
<form action="/search" method="post" id="forrm-contact">
  <div class="forrm-row">
    <label for="forrm-contact-name">Name*</label>
    <input type="text" id="forrm-contact-name" name="forrm-contact-name" placeholder="Forename Surname" value="" aria-required="true" required>
  </div>
  <div class="forrm-row">
    <label for="forrm-contact-email">Email*</label>
    <input type="email" id="forrm-contact-email" name="forrm-contact-email" placeholder="name@example.com" value="" aria-required="true" required>
  </div>
  <div class="forrm-row">
    <label for="forrm-contact-message">Message*</label>
    <textarea id="forrm-contact-message" name="forrm-contact-message" placeholder="Your message here..." value="" aria-required="true" required></textarea>
  </div>
  <input type="submit" value="Search">
</form>
//Include minfied version and instantiate form
<script src="js/forrm.min.js">
<script>Forrm.init('#forrm-contact', {listMessages: true});
